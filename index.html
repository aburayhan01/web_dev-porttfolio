Vanilla JavaScript

Vanilla JavaScript refers to using plain, pure JavaScript without any libraries or frameworks. It is lightweight, efficient, and ideal for building simple or highly customized solutions directly in the browser.

react

React is a popular JavaScript library for building interactive user interfaces. It uses a component-based architecture and a virtual DOM to optimize rendering and improve performance.React is often used for dynamic, single-page applications.

Node.Js

Node.js is a JavaScript runtime built on Chrome's V8 engine. It enables running JavaScript on the server side, making it ideal for building scalable, event-driven, and non-blocking server-side applications.

mongodb

MongoDB is a NoSQL database that stores data in flexible, JSON-like documents. It is highly scalable and suitable for modern applications requiring high performance and adaptability to rapidly changing data structures.
